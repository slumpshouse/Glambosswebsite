# Glam Goddess Shop - Full App Wireframe

## System Flow Diagram

```
CUSTOMER FLOW:                          ADMIN FLOW:
Home → Cart → Checkout → Payment    →   Login → Dashboard → Products/Customers/Sales/AI
  ↓                                         ↓                    ↓
Browse Products                      Admin Operations       Manage Everything
Add to Cart                           View KPIs               Logout
  ↓                                        ↓
Enter Customer Info                  Quick Actions
Phone Auto-Format
  ↓
Stripe Payment UI
  ↓
30s Success Message
  ↓
Redirect Home
```

## Key Features Summary

| Feature | Location | Purpose |
|---------|----------|---------|
| Product Browse | Home | Customers discover available products |
| Add to Cart | Home | Build shopping cart with quantities |
| Checkout | /checkout | Enter customer info, review order |
| Auto-format Phone | Checkout/Manual Sale | Normalize phone to 123-456-7890 |
| Stripe Payment | /payments/* | Secure card processing |
| Admin Dashboard | /admin/dashboard | KPI overview, quick actions |
| Product CRUD | /admin/products | Create, edit, delete products |
| Customer Search | /admin/customers | Find customers, view metrics |
| Sales List | /admin/sales | View all transactions |
| Manual Sale | /admin/sales/manual | Log direct sales with preview |
| AI Summary | /admin/ai-summary | Generate weekly insights |
